# Daleen 2K BOT

The ultimate multi-bot session manager for NBA 2K on Xbox Cloud Gaming (xCloud). Run up to 9 bots simultaneously — each with its own Xbox account, its own game session, and full gamepad control.

## Features

- **9 Simultaneous Bots** — each running in an isolated browser session
- **Auto-Launch** — bots find and start your selected game instantly
- **Park Bot** — auto-walks to stay active + grabs Got Next automatically
- **Game Selection** — NBA 2K26 and NBA 2K25 supported out of the box
- **Live Ping Monitor** — real-time latency per bot (color-coded green / yellow / red)
- **Full Bot Controls** — fullscreen, minimize, close, and reload per tile
- **Keyboard Shortcuts** — Arrow keys / WASD to navigate, F / M / Delete for quick actions
- **Auto-Updater** — downloads new versions in the background with SHA-256 verification
- **License System** — hardware-bound keys, one activation per machine

## What You Need

- **Windows 10 or 11** (64-bit)
- **8 GB RAM** minimum, 16 GB recommended
- **Stable internet** — at least 25 Mbps per active bot
- **Up to 9 Xbox Game Pass Ultimate accounts** — one per bot. Each account needs its own active XGPU subscription to access Xbox Cloud Gaming.
- **NBA 2K25 or NBA 2K26** — the game must be purchased separately on each Xbox account. Cloud Gaming streams the game, but you still need to own it.

> No GPU required. All game rendering happens on Microsoft's cloud servers — your PC just receives the video stream.

## Quick Start

1. Run the installer and open the app
2. Enter your username and paste your license key
3. Pick your game (2K26 or 2K25) and set your bot count
4. Hit **START BOTS** — each bot auto-launches the game
5. Sign into a different Xbox account in each bot window
6. Your logins are saved — next time it's one click and go
