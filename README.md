# Daleen 2K BOT

A professional multi-bot session manager for NBA 2K on Xbox Cloud Gaming (xCloud).

## Features

- Launch up to 9 simultaneous bots, each with isolated Xbox accounts
- Auto-launch — bots automatically find and start your selected game
- Park Bot — auto-walks (anti-AFK) + A-button spam (grabs Got Next)
- Game selection — NBA 2K26 and NBA 2K25 supported
- Real-time ping monitoring per bot (color-coded good / fair / poor)
- Fullscreen, minimize, and close controls per bot tile
- Keyboard shortcuts — Arrow keys / WASD to switch bots, F / M / Delete
- Built-in auto-updater with SHA-256 integrity verification
- License key system with hardware binding
- Secure input validation on all IPC channels

## Requirements

- Windows 10/11 (64-bit)
- 8 GB RAM minimum (16 GB recommended)
- Stable internet — 25+ Mbps per active bot
- Each bot needs its own Xbox account with Game Pass Ultimate

## Download

Get the latest version from the [Releases](https://github.com/DaleenNhodes/daleen-2k-bot-releases/releases) page.

## Setup

1. Run the installer
2. Enter your username and license key
3. Pick your game, set bot count, hit START
4. Sign into a different Xbox account in each bot
5. Done — logins save automatically
