# Daleen 2K BOT

A professional multi-bot session manager for NBA 2K26 on Xbox Cloud Gaming (xCloud).

## Features

- Launch up to 9 simultaneous bots, each with isolated Xbox accounts
- Auto-launch — bots automatically navigate to NBA 2K26 and start streaming
- Auto Move — keeps all players moving to prevent AFK timeout
- Get On — auto-presses A to grab Got Next after a loss
- Real-time ping monitoring per bot (color-coded good / fair / poor)
- Fullscreen, minimize, and close controls per bot tile
- Built-in auto-updater with integrity verification
- License key system with hardware binding
