# Daleen 2K BOT

Multi-session bot manager built for NBA 2K on Xbox Cloud Gaming.

---

## What It Does

Runs up to 9 bot instances at once — each in its own isolated session with separate Xbox credentials. Auto-launches your game, keeps sessions alive, and handles Got Next queuing without manual input.

## Highlights

- 9 concurrent sessions with full isolation
- One-click game launch per session
- Park mode — movement + queue automation in one toggle
- Dual game support (2K26 / 2K25)
- Per-session latency monitoring
- Tile controls — resize, minimize, close, reload
- Background updates with integrity checks
- Hardware-locked licensing

## Before You Buy

- **Windows 10/11** required
- **9 Xbox Game Pass Ultimate subscriptions** — one per bot session
- **NBA 2K** must be owned on each account (not included with Game Pass)
- Stable connection recommended — 25 Mbps+ per active session
- No GPU needed — rendering is server-side
